Hit the API endpoint with the first 5 characters of the SHA-1 hash you wish to check.

A1B2C

You will receive back all the hashes matching that prefix in the following format.

[
  {
    "hash":"A1B2C00911E3E9DB11771EE606591222AE43D59C",
    "count":13
  },
  {
    "hash":"A1B2C00989E1C23D9D4F0EB6CE471759873B051B",
    "count":4
  },
  {
    "hash":"A1B2C00A25C62845E734E76494B9382E95C0A3BB",
    "count":3
  }
]
 The amount of items in the response has been trimmed.
